# 📚 Study Tracker

A simple Python-based Study Tracker that helps users record study sessions, calculate total study hours, analyze subject-wise progress, and manage study records through an interactive command-line interface.

## Features

* Add new study sessions
* View all recorded study sessions
* Calculate total study hours
* Analyze subject-wise study time
* Delete unwanted study sessions
* Input validation using exception handling

## Technologies Used

* Python
* Lists
* Dictionaries
* Functions
* Loops
* Exception Handling

## How It Works

The application stores study sessions in a list of dictionaries.

Each session contains:

* Subject Name
* Hours Studied

Example:

```python
{
    "subject": "Python",
    "hours": 2.5
}
```

The program allows users to:

1. Add study sessions
2. View all sessions
3. Calculate total study hours
4. Analyze study time by subject
5. Delete a session
6. Exit the application
